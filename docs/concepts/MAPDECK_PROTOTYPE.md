# MapDeck 試作（採用前）

遠征マップで **ホップ射程の射撃** を一度通すための縦切り。収納術式の二本デッキや三軸廃止はまだ正式採用していない。

## 起動

1. Unity Play
2. F10 → 開発者 → **遠征マップ**（または通常遠征開始）
3. 画面下に **MAP DECK** 手札バーが出る

## 操作

1. 手札の **石弓（射程2）** または **投石（射程1）** を選ぶ
2. 地図上で **シアンに光った地点**（届く敵／経路）を確認
3. **敵のいる地点**をタップ → マップ上でダメージ
4. HP0で撃破（戦闘卓スキップ）
5. 隣接ノードに生きた敵がいる状態で踏み込む → 従来どおり戦闘卓
6. **ターン補給**で EN と手札を補充

## 配置（外郭スパイン）

| 地点 | 内容 |
|---|---|
| 入口(4) | 開始 |
| 23 → 11 → **14 橋守** → 20 → **10 巡回兵** | 試作ルート |

敵トークンは紫の駒＋HP数字。

## コード

- `MapCombatSystem.cs` — ホップ距離・射線・Strike
- `ExplorationRunState.mapHand / mapEnemies` — 戦闘手札とは分離
- `ExplorationMapStage` — 敵駒・シアン射程ハイライト
- `PackspireUiFoundation.ExplorationMap` — 手札バー
