# 遠征＝封印格子盤（正ルート）

写実の一枚絵＋ノード地図は捨て、**均等マスの盤ゲーム**として遠征を見せる。  
戦闘も同じ画面骨格で行う（遷移しない）。全体の進行は旧ノード遠征と同型。

## レイアウト

- 左ドック：状況テキスト。導線中だけ矢印／一手戻す／進む。ゲート発見時は進出／帰還の選択
- 右ステージ：マップ（戦闘時は敵立ち絵）
- 右下：カード扇。戦闘時は常時オープン
- 扇の下：ENレール（戦闘時は SKILL / END TURN）

## 移動サイクル

1. 配置 → 導線 → そのルート進行（戦闘／イベントは割り込み）
2. ルート終端 → 手札・EN補充 → いまの位置からまた配置

## エリア区切り

- ダンジョンは複数区画（サイズ・敵数は区画ごとに変化）
- マス **次**＝次区画への裂け目（発見後に進出／まだ探索を選択）
- マス **帰**＝帰還点（発見後に帰還／まだ探索を選択）
- 最終区画には「次」がなく、「帰」で持ち帰り
- 上バー **撤退**＝途中撤退（戦利品持ち帰り可）。正式クリアは帰還点

## マス

| 記号 | 意味 |
|------|------|
| 敵 | 同画面戦闘 → Reward |
| 異 | Event 画面 → ルート続行 |
| 次 | 次区画選択 |
| 帰 | 帰還選択 |

敵・イベント・ゲートの座標は入場／区画展開ごとにランダム。数は区画定義。

## Unity

- `GridBoardSystem.Create(dungeonId)` → 区画0を展開
- `TryAdvanceArea` → 次区画（盤再生成）
- `UiConfirmGridReturn` → `FinishRun(true)`
