# PACKSPIRE Prototype

装備・敵・役職・勢力・ダンジョン・数値・実装済み仕様は [GAME_DATA.md](GAME_DATA.md) を参照してください。

バックパック配置とデッキ構築を組み合わせた、1ラン5～8戦のローグライク・カードバトル試作版です。

未実装・保留中の構想と今後の優先順位は [UNIMPLEMENTED_IDEAS.md](UNIMPLEMENTED_IDEAS.md) にまとめています。ノートはこのREADMEを含めた3ファイルへ統合しています。

## 起動

初回起動時は、男性系2タイプ・女性系2タイプと髪型3種から探索者の外見を作成します。作成した外見は拠点中央と戦闘画面へ反映されます。

`index.html` をブラウザーで開くだけで遊べます。ローカルサーバーを使う場合は、このフォルダーで次を実行してください。

```powershell
python -m http.server 8000
```

その後、`http://localhost:8000` を開きます。

## 遊び方

1. 所持品の形を確認し、6×4のバックパックへドラッグして配置します。クリックして空きマスを選ぶ操作も使えます。
2. 紫色のルーンを武器に隣接させると、その武器が生成する攻撃カードが強化されます。
3. 生成カードが4枚以上になったら戦闘を開始できます。
4. 敵の次の行動を見ながら、3エネルギー以内でカードを使用します。
5. 勝利後に戦利品を一つ選び、ダンジョンごとの最終戦にいるボス撃破を目指します。

戦闘中は各ターン1回、1エネルギーを払って「荷物整理」ができます。配置を確定すると、変更後の装備から山札と手札が再構築されます。

配置済みアイテムはドラッグして移動できます。所持品または配置済みアイテムを選択し、「↻ 回転」を押すと90度回転します。右クリックで一つだけ取り外せます。「すべて外す」で荷造りをやり直せます。

## 配置ルール

- バックパックは6×4マスです。装備の形状と空きマスだけが収納制限になります。
- 剣・盾・短剣・ルーン・結晶は隣接する装備によってカードが強化されます。
- 薬草は上端、胸当ては下端、爆弾は右端に置くと追加効果を得ます。
- 発動中の装備はバックパック上で発光し、右側の生成デッキ欄に `LINK!` として表示されます。
- 灰色で表示される条件は、現在まだ発動していない配置効果です。

## 探索

戦闘後は次の経路を三つから選びます。

- 未知の物音：選択肢によって結果が変わるランダムイベント
- 行商人：所持金で新しい装備を購入
- 野営地：HPを回復
- 崩れた保管庫：HPを失う代わりに所持金を獲得

敵は階層ごとの候補からランダムに出現します。装備は全12種類、通常敵は6種類、最終ボスは1種類です。

## 永続強化と秘密

ゲームは「拠点」と「遠征」に分かれます。保管庫の装備で永続バックパック編成を作り、ダンジョン選択後は道具袋・地図断片・契約を確認して遠征します。踏破・途中帰還では所持金を持ち帰り、敗北時は未確保の戦利品と遠征中のゴールドを失います。

拠点ホームでは遠征・荷造り・保管庫・施設・ステータス・図鑑・記憶庫を選びます。「荷造り」では3つの編成セットを保存でき、セットごとにバックパック種類、装備配置、採用する装備カード8枚を保持します。遠征準備で使用セットを選び、「行き先選択→編成・道具・契約確認→探索開始」の順に進行します。

装備は死亡しても失われず、バックパック編成も維持されます。家宝に指定した1個だけは使用回数、戦闘・ボス・敗北履歴と傷跡を記録し、通常装備は個体色・特性・強化段階だけを保存します。安全マスと保管装備の持込上限は廃止しました。

ダンジョンは「古塔パックスパイア」全5戦、「灰熱の鋳造坑」全6戦、「虚ろなる大記憶庫」全8戦です。踏破するたびに次の高難度ダンジョンが恒久解禁されます。

バックパックの各マスには火・水・風・土の色があります。装備個体のセル色と特性は入手時に決まり、その後は固定されます。配置した全装備がカード候補を出し、その中から装備カードを最大8枚選びます。装備セルと真下のマスが同色ならバックパック全体の色数へ加算され、全体色数は全カードと個体特性を強化します。それとは別に、装備同士を隣接させるとLINK効果や合成カードが発動します。基本職業カード4枚は固定でデッキへ入ります。

戦闘バックパックの配置は遠征終了後も保存され、次回遠征で復元されます。敵遭遇時や戦闘中の全面整理は行いません。ダンジョンで発見した装備は未鑑定の戦利品袋へ入り、即座には装備できません。踏破時に最大2個を選び、拠点へ持ち帰って鑑定します。道具袋は初期5枠で、回復薬などをカードとエネルギーを消費せず戦闘中に使えます。消耗品は行商人から補充できます。

画面右上の「記憶庫」から、周回をまたいで保存される進行状況を確認できます。データはブラウザーのローカルストレージへ自動保存されます。

- 欠けた剣のカードを100回使うと「黒鉄の百斬剣」へ恒久進化し、以降のランでも強化状態で出現します。
- 連撃の短剣と折畳み槍にも、それぞれ異なる使用回数で恒久進化があります。
- 挑戦回数、踏破回数、カード使用、回転、購入、イベント、戦闘中整理を永続記録します。
- 全16個の秘密条件があります。未発見の条件と報酬は記憶庫でも伏せられています。
- 秘密の報酬には初期HP・初期ゴールドなどの恒久強化があります。
- 敗北しても武器熟練度と秘密の発見状況は失われません。

## 役職と転職

ラン開始時に役職を選択します。最初から選べる基本職は4種類です。

- 戦士：武器攻撃とHPが得意
- 守護兵：防御と最大HPが得意
- 斥候：短剣・槍と所持金が得意
- 工匠：爆弾、薬草、フラスコなどの道具が得意

戦闘に勝つと現在職の役職レベルが上昇します。継職の祭壇では役職変更、修練、条件を満たした上級職の解禁ができます。最終戦の直前には継職の祭壇が必ず候補へ現れます。

通常の転職では発見できない隠し職も4種類あります。解禁条件には以下の系統がありますが、正確な配置はゲーム内では伏せられています。

- 特定の武器種を一度に収集する
- 特定の防具構成をバックパックへ配置する
- ルーンをバックパックの特殊な位置関係で結ぶ
- 異なるアイテム種を幾何学的に配置する

役職レベルと現在職は周回をまたいで恒久保存されます。役職選択は初回だけで、変更はダンジョン内の継職の祭壇でのみ行えます。戦闘勝利と祭壇の修練で現在職がレベルアップします。習得済みの全役職効果が同時に働き、全役職の合計レベル上限は100です。

色一致に特化した複合職として、風一致の「瞬刃士」、土一致の「定錨騎士」、複数色一致の「攻城導師」があります。また、右2列へ片手武器だけを構えることで隠し職「片手剣鬼」を解禁できます。

敗北から回収された装備には、長所と短所を持つ傷跡が最大3個まで刻まれます。拠点の保管庫は初期4枠で、保管金を使って最大16枠まで拡張できます。

高密度なアニメ・ダークファンタジー調のイラストをゲーム内へ組み込んでいます。

- `assets/equipment-sheet.png`：全12種類の装備

## 追加ゲームシステム

- バックパック選択：標準、行商人、魔導、棺型、生きている鞄の5種類
- 装備合成：剣＋熾火、盾＋結晶、爆弾＋フラスコで隠しカード生成
- 方向接続：配置アイテムの矢印方向へルーン効果が流れる
- 耐久度：装備カードの使用で低下し、破損すると効果半減。野営地で修理
- 装備個体：同じ種類でも耐久度と使用回数を個別に記録。将来の家宝・進化・傷・保険にも対応できる形式
- ライバル探索者：決闘または取引を選ぶイベント
- 収納攻撃：敵が装備を妨害し、次ターンのカードコストを増加
- 位置カード：上段の武器と端の防具を参照する「陣形戦術」
- 呪い装備：強力だが取り外せず、周囲のコストを増加
- 協力勢力：鉄殻軍、胞子教団、荷造り師組合から選択。虚無の巡礼者は隠し勢力
- 周回契約：敵攻撃、行動予告、最大HP、戦闘中整理へ制約を付けて報酬増加
- 噂図鑑：カード使用回数に応じて、合成や隠し職のヒントが開示

遠征準備画面で、協力勢力、バックパック、家宝、複数の周回契約を選択できます。勢力はランダムではなく任意選択で、勝利による貢献度から階級と効果が成長します。階級3では勢力限定職も解禁されます。
